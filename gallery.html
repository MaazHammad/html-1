HTML5: Figure and Figure Caption

<figure> </figure>

image often come with captions.HTML5 has introduced a new element to contain image and their caption so that the two are associated.
You can have more than one image inside the element as long as they all share the same caption.

<figcaption> <figcaption>

The element has been added to HTML5 in order to allow web page authors to add a caption to an image.
Before these element were created there was no way to associate an element with its caption.\
Order browsers that do not understand HTML5 element simply ignore the new element and display the contact of them.
